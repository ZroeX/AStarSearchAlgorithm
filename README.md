# AStarSearchAlgorithm
AStar Search Algorithm Project.

Left mouse button : Set node close/open.  
"R" key : Reset map.  
"Space" key : Find path.  
  
Node Color :  
Green : Start node.  
Gray : End node.  
White : Open node.  
Black : Close node.  
Blue : Checked node.  
Red : Final path.
